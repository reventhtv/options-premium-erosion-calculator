BLACK-SCHOLES-MERTON MODEL INFO
Model Parameters:
  • S = Spot Price (₹)
  • K = Strike Price (₹)
  • T = Time to Expiry (years)
  • r = Risk-free Rate (%)
  • σ = Implied Volatility (%)
  • q = Dividend Yield (%)
Key Formulas:
  • d₁ = [ln(S/K) + (r - q + σ²/2)T] / (σ√T)
  • d₂ = d₁ - σ√T
  • Call = S⋅e⁻ᵠᵀ⋅N(d₁) - K⋅e⁻ʳᵀ⋅N(d₂)
  • Put = K⋅e⁻ʳᵀ⋅N(-d₂) - S⋅e⁻ᵠᵀ⋅N(-d₁)
