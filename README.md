# Options Premium Erosion Calculator

A professional web tool for calculating theta decay (premium erosion) of call and put options.

## Features

- ✅ Calculate daily/weekly/total premium erosion
- ✅ Separate calculations for Call and Put options
- ✅ Interactive visualization with Chart.js
- ✅ Advanced settings (theta acceleration, volatility impact)
- ✅ Responsive design with Bootstrap 5
- ✅ Export results functionality

## Live Demo

[https://reventhtv.github.io/options-premium-erosion-calculator/](https://reventhtv.github.io/options-premium-erosion-calculator/)

## How to Use

1. Enter option details (strike, premium, Greeks)
2. Click Calculate for Call/Put
3. View erosion projections and charts
4. Adjust advanced settings as needed

## Input Parameters

| Parameter | Description |
|-----------|-------------|
| Theta (θ) | Daily time decay |
| Gamma (Γ) | Rate of delta change |
| Delta (Δ) | Price sensitivity |
| Vega (ν) | Volatility sensitivity |
| Days to Expiry | Time until expiration |
